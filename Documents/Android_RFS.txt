Android RFS (Root File System)
 - cache 
 - data : application data and download data...
 - dev : char, block,, devices
 - proc : kernel, process information.
 - sys : device driver information.
 - sbin : ueventd exec files.
 - sdcard : link file. (/mnt/sdcard/)
 - system
   - app : default APK folder.
   - bin : deamon, bin, toolbox..files.
   - framework : java libraries.
   - lib : Native libraries (*.so)
     - hw : HAL modules (*.so)
   - modules : Kernel Modules. (*.ko)
   - vendor : vendor's HAL (*.so)
